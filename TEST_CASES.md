# VReport 测试用例

> 覆盖范围：报表设计、公式编辑、预览正确
> 适用阶段：功能测试 / 回归测试 / 冒烟测试

---

## 用例标识规则

- `DS`：Designer（报表设计）
- `FE`：Formula Editor（公式编辑）
- `PV`：Preview（预览）
- `EX`：Export（导出）

优先级：`P0` 阻塞级，`P1` 高，`P2` 中，`P3` 低

## 四、冒烟测试组合

### 组合 1：销售报表端到端

1. 新建报表
2. 创建 JSON 数据源：`[{region:'华北',sales:100},{region:'华南',sales:200}]`
3. 创建数据集 ds1
4. A1 输入「区域」，B1 输入「销售额」
5. A2 输入 `${ds1.region}`，设置主格为 A2，方向向下
6. B2 输入 `${ds1.sales}`，主格为 A2
7. A3 输入「合计」，B3 输入 `=sum(B2:B2)`
8. 预览 → 确认两行数据 + 合计正确
9. 导出 Excel 确认内容一致

### 组合 2：参数过滤报表

1. 创建带 region 字段的数据源
2. 创建参数 regionParam，默认值为「华北」
3. 设计报表展示 region 和 sales
4. 预览 → 默认只显示「华北」
5. 切换参数为「华南」→ 数据刷新
6. 清空参数 → 显示全部

### 组合 3：复杂公式与条件格式

1. 设计表格含 score 字段
2. C2 输入 `=if(${ds1.score}>=60,"及格","不及格")`
3. 设置条件格式：score<60 红色背景
4. 预览 → 不及格行背景变红，文本正确
